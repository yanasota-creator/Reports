# INDEX

このリポジトリ内ファイルへのリンク一覧．このファイルのraw URLだけClaudeに送れば，
ここに書かれているリンクを辿って各ファイルを読み込める．

（注）リポジトリ名やファイルを追加・変更したら，このファイルも更新すること．
（注）リンクは必ず `raw.githubusercontent.com/.../main/...` の形式にすること．
　　　`github.com/.../blob/main/...` の形式だと読み込めない．
